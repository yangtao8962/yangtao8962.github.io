# Docker可视化工具

# Portainer

官方安装说明：https://www.portainer.io/installation/ (opens new window)

# 安装

docker pull portainer/portainer

docker run -d -p 8000:8000 -p 9000:9000 --name=portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer

如果时远程管理，需要修改将要被远程连接的客户机的docker.service 文件，开通docker的远程管理：

vim /lib/systemd/system/docker.service

ExecStart=/usr/bin/dockerd -H unix:///var/run/docker.sock -H tcp://0.0.0.0:2375

运行以上命令后就可以通过web界面管理docker了，地址http://120.79.87.184:9000/#/init/admin

注册，登录后就可以通过图形化界面管理docker了

# 建议

将portainer添加到docker-compose.yml中

version: "3.2"

services:
  portainer:
    image: portainer/portainer
    container_name: portainer
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - portainer_data:/data
    networks:
      - hello
    ports:
      - "8000:8000"
      - "9000:9000"

volumes: 
  portainer_data:
